CodeBook
==================

This file describes the data and changes made for this project.

The output can be found in output.txt. It is a R data frame written as a table to a txt file. 
The variable names are all the ones given in the project, 
except for activity is used as a header for the column that describes the activity, 
and subject is used as a header for the column with the subject number.

All changes and operations done are explained in run_analysis.R and README.md.
